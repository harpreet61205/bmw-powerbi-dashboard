# BMW Cars Analysis Dashboard

## Description
This project is an interactive dashboard created using Power BI to analyze BMW cars data and generate meaningful insights through data visualization.

## Technologies Used
- Power BI
- Microsoft Excel
- Data Visualization
- DAX

## Features
- Interactive dashboard with filters and slicers
- Visualization of car models, prices, and performance insights
- Easy exploration of BMW car data through charts and KPIs

## Purpose
The purpose of this project is to analyze BMW cars data and present insights in a clear and interactive way using Power BI dashboards.
